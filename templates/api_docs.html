🏠 Home Assistant Integration
Add these to your configuration.yaml. Replace YOUR_KEY with your API key and {{ base_url }} with your Jen URL.
Presence Detection (device tracker)
Detects if a device is home based on whether it has an active DHCP lease. Replace the MAC with your phone's MAC address.
# configuration.yaml
rest:
  - resource: "{{ base_url }}/api/v1/devices/aa:bb:cc:dd:ee:ff"
    headers:
      Authorization: "Bearer YOUR_KEY"
    sensor:
      - name: "Phone Presence"
        value_template: "{{ '{{' }} 'home' if value_json.online else 'not_home' {{ '}}' }}"
        icon: mdi:cellphone
Subnet Utilization Sensor
# configuration.yaml
rest:
  - resource: "{{ base_url }}/api/v1/subnets"
    headers:
      Authorization: "Bearer YOUR_KEY"
    sensor:
      - name: "Production DHCP Utilization"
        value_template: "{{ '{{' }} value_json.subnets[0].utilization_pct {{ '}}' }}"
        unit_of_measurement: "%"
        icon: mdi:percent
Kea Health Check
# configuration.yaml
rest:
  - resource: "{{ base_url }}/api/v1/health"
    binary_sensor:
      - name: "Kea DHCP"
        value_template: "{{ '{{' }} value_json.kea_up {{ '}}' }}"
        device_class: connectivity
