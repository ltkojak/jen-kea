📥 Import / Restore

⛔ Destructive operation. Importing will replace existing data in the selected tables. This cannot be undone. Make a backup first.

Upload a Jen export file (.json.gz). Jen will read the file, show you exactly what's in it, and ask you to confirm before touching any data. The file header identifies whether it's a Jen or Kea export — you cannot accidentally import the wrong one.

Must be a file exported from Jen's Database page. Files from other tools are not supported.
